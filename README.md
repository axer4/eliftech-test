To start project on your local machine: 
1) Clone this repo
2) open with code editor
3) npm i
4) json-server --watch db.json  - to start local database
5) src/redux/bankOperatons.js - change axios.default.baseURL = 'https://yourlocalhost:port'
6) npm start
